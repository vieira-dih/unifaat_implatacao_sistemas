# Aula 015 - Projeto Prático: Pipeline de CI/CD do Zero

**Disciplina:** Implementação de Sistemas  
**Curso:** Análise e Desenvolvimento de Sistemas - UniFAAT  
**Professor:** Alexandre Tavares  

## 🎯 Objetivos da Aula

- Construir pipeline CI/CD completo do zero
- Integrar GitHub Actions com AWS
- Implementar testes automatizados
- Automatizar deploy para produção
- Aplicar todos os conceitos do curso

## 📚 Conteúdo Programático

### 1. **Pipeline CI/CD Completo**
- Integração contínua com GitHub Actions
- Testes automatizados (unit, integration, e2e)
- Build e containerização
- Deploy automático para AWS

### 2. **GitHub Actions Avançado**
- Workflows complexos
- Secrets e variáveis de ambiente
- Matrix builds
- Conditional deployments

### 3. **Integração com AWS**
- Deploy para EC2/ECS
- Integração com RDS
- S3 para assets estáticos
- CloudWatch para monitoramento

### 4. **Projeto Integrador**
- Aplicação completa (Frontend + Backend + Database)
- Pipeline end-to-end
- Monitoramento e alertas
- Documentação técnica
